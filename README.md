# Estimate Resource Creator

### Setup
```bash
# Run the command
# Installs clasp (Command Line Apps Scripts Projects) and other project dependencies
pnpm install
```

### Build & Push to the Cloud
```bash
# Build 
pnpm build

# Build and push combined 
pnpm push
```

### Run Tests
```bash
pnpm test
# OR
pnpm vitest
```
